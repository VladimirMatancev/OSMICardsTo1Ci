Справочник "Источник данных"

В данном справочнике хранятся описания способов получения данных для передачи на карты клиетов.

  • Объект конфигурации - Объект метаданных конфигурации, из которого будут браться данные
  • Поле основного отбора - Поле даннх объекта метааднного, в котором хранится ссылка на записываемый обект
  • Поле - Поле даннх объекта метаднного, в котором хранитя ссылка на карту клиента
  • Табличная часть - Имя табличной части (если нужно), в котором хранится ссылка на карту клиента
  • Тип регистра - указывается брать конкрутную запись, или обороты/остатки/срез последних по регситру (если регистр поддерживает такое)
  • Текст программы - данное поле содержит сгенерированный на основании вышеуказанных полей код, исполняемый для полчения списка карт, по которым будет отрабатывать данная настройка. Данный текст можно редактировать
  • Табличная части "Отборы" содержат в себе дополнительные отборы для данного события (параметр - имя отбираемого поля, значение - значение с которым будет сравниваться данное поле)
  • Параметр - имя сравниваемого параметра
  • Значение - значение, с которым сравнивается параметр
  • ТипСравнения - тип сравнения

    • Действие - описание действия, которое нужно произвести с картами при наступлении данного события

    Текст заголовка поля, Текст пуш сообщения, Текст поля - данные поля отправляются на карту в и зависимости от настройек создается динамическая окция, либо отправляется обыкновенное пуш сообщение с обновлением поля.
