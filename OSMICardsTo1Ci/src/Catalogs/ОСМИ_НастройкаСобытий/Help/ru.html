Справочник "Настройка событий"

В данном справочнике хранятся настройки событий, по которым при наступлении события (проведения конкретного документа/ записис справочника) будут обновляться данные на картах телефонов у клиентов. будет отправляться пуш сообщение.

  •  Объект конфигурации - Объект метаданных конфигурации, при записи/проведения которого будет срабатывать событие

  • Поле основного отбора - Поле данных объекта метаданного, в котором хранится ссылка на записываемый объект

  • Поле - Поле данных объекта метаданного, в котором хранится ссылка на карту клиента

  • Табличная часть - Имя табличной части (если нужно), в котором хранится ссылка на карту клиента

  • Табличная части "Отборы" содержат в себе дополнительные отборы для данного события (параметр - имя отбираемого поля, значение - значение с которым будет сравниваться данное поле).

  • Тип регистра - указывается брать конкретную запись, или обороты/остатки/срез последних по регистру (если регистр поддерживает такое)

  • Текст программы - данное поле содержит сгенерированный на основании вышеуказанных полей код, исполняемый для получения списка карт, по которым будет отрабатывать данная настройка. Данный текст можно редактировать.

  • Действие - описание действия, которое нужно произвести с картами при наступлении данного события

  • Текст заголовка поля, Текст пуш сообщения, Текст поля - данные поля отправляются на карту в и зависимости от настройки создается динамическая акция, либо отправляется обыкновенное пуш сообщение с обновлением поля.

Табличная часть справа отображет структуру объекта метаданных
