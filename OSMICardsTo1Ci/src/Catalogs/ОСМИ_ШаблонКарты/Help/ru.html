Справочник "Шаблон карты"

 

Данный справочник содержит в себе информацию о шаблонах карт, сопоставление полей шаблона и данных из 1С.
Содержимое баркода - содержит информацию о том, что отобрается в баркоде на карте клиента. 
Способ получения содержимого баркода - ссылка на источник данных, с правилами получения данных. 
Отображать баркод - отображать баркод или нет 
Содержимое подписи - содержит информацию о том, что отобрается в подписи баркода на карте клиента. 
Способ получения содержимого подписи - ссылка на источник данных, с правилами получения данных
Отображать подпись - отображать подпись баркода или нет 
 
Табличная часть "Поля шаблона"
  • Имя - имя поля шаблона (соответствует имени поля карты)
  • Способ получения - способ получения данных для заполнения (справочник Источник данных)
  • Текст оповещения - Текст пуш сообщения, отправляемый при обновлнии данного поля (если не указан - отправляется стандартный текст сообщения)
  • Отправлять ПУШ - отправлять или нет пуш сообщение при обновлении данного поля.
