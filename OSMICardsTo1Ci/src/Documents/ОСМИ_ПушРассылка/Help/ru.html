Отправка пуш сообщений

Для создания пуш рассылки нужно перейти на вкладку ОСМИ, и перейти по навигационной ссылке «Пуш рассылка», после чего откроется список документов «пуш рассылка».

Для отправки сообщения клиентам нужно создать новый документ, указать желаемую дату отправки, указать сообщение (желательно не очень длинное, чтобы оно все было отображено на экране телефона), выбрать список получателей и провести документ.

Для проверки статуса доставки можно открыть по кнопке на форме «проверить доставку» форму проверки доставки.

В ней отображается список карт клиентов, на чьи телефоны сообщение было доставено.
